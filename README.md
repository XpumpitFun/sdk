# Xpumpit

Xpumpit is your instant AI agent for Twitter and web interactions. Tag our bot in replies or mention it in a thread, and it will launch, monitor, and query automatically — making engagement smarter and easier than ever.

## Features

- **Instant AI Response** – Launch our AI agent immediately by tagging it.
- **Thread Monitoring** – Start a thread and let Xpumpit monitor and respond automatically.
- **Smart Query Handling** – Ask questions, and Xpumpit will process and provide intelligent responses.
- **Cross-Platform Links** – Works seamlessly via [Xpumpit on X (Twitter)](https://x.com/XpumpitFun) and [Xpumpit Web](https://xpumpit.fun/).

## How to Use

1. **Tag the Bot**  
   Reply to any tweet with a tag to `@XpumpitFun` and the AI agent will activate.  

2. **Start a Thread**  
   Mention `@XpumpitFun` in a new thread. The agent will monitor and engage automatically.  

3. **Query the AI**  
   Simply type your question or request after tagging the bot. Xpumpit handles the rest.
